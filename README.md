# Python Chess Game ♟️

A fully playable chess game developed in Python using Pygame.

## Features

- Legal movement for all chess pieces
- Turn-based gameplay
- Check detection
- Checkmate detection

## Technologies Used

- Python
- Pygame

## Skills Demonstrated

- Algorithm Design
- Problem Solving
- Event-Driven Programming
